This code resolves all sodokus available at http://www.sudoku.com/ (easy to expert).
The goal here is to rewrite the Java version I created a week ago in C++.
It also has some performance enhancements.

The algorithm was created by me, using my personal knowledge on how I would resolve it.
There must have better solutions out there on how to calculate the right answer - I'll try different approaches in the future.

Resolved 100.000 within 22.402ms
CPU: Intel i7 2600, VMWare with 2 cores, 2GB, Windows 7 64.
